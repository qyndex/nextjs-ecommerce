# E-commerce Store

Full e-commerce storefront with product catalog, shopping cart, and Stripe checkout. Built with Next.js, shadcn/ui, and Stripe.

**Stack:** Nextjs | **Complexity:** complex
**Tags:** `ecommerce`, `store`, `stripe`, `cart`, `checkout`, `products`, `shadcn`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
